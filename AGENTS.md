This project is used to study transforms that may improve cuts on polytopes in a mixed-integer programming context. Most files are unrelated to each other. 

Files ending in loader.py are for loading different types of MIPs.

Whenever you run Python files or Python code of any kind you need to do it inside the virtual environment selected for VSCode. (Use the .venv folder that is a soft link to a different folder.)

Tests are to be put in the tests subfolder and made runnable with pytest.

Don't generate a 'reference guide,' 'comprehensive summary document,' or any kind of documentary md files until the user explicitly requests them.
